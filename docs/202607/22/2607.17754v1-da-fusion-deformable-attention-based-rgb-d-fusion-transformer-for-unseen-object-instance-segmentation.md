## 摘要
在物流自动化中，对未知物体的精确分割对于在杂乱环境中进行高效机器人操作至关重要。诸如料箱拣选和货架拣选等任务需要鲁棒的感知能力来处理遮挡、多样化的物体形状以及复杂的空间布局。传统的基于RGB的方法由于依赖纹理而倾向于过度分割物体，而基于深度的方法则因主要关注几何特征而常常分割不足。为解决这些局限，我们提出了DA-Fusion，一种基于可变形注意力的RGB-D融合Transformer，专为未知物体实例分割设计。DA-Fusion有效结合了RGB和深度数据的优势，提高了在杂乱和多层物体环境中的分割精度。我们还引入了Object Clutter Bin Dataset (OCBD)，这是一个专门为评估自上而下视角的料箱拣选场景而设计的基准数据集。大量评估表明，DA-Fusion在不同环境中均优于最先进的方法，使其特别适用于现实世界的物流任务。

## Abstract
In logistics automation, precise segmentation of unseen objects is crucial for efficient robotic manipulation in cluttered environments. Tasks such as bin-picking and shelf-picking require robust perception to handle occlusions, varying object shapes, and complex spatial arrangements. Traditional RGB-based methods tend to over-segment objects due to their reliance on texture, while depth-based methods often under-segment by focusing primarily on geometric features. To address these limitations, we propose DA-Fusion, a deformable attention-based RGB-D fusion Transformer designed for unseen object instance segmentation. DA-Fusion effectively combines the strengths of both RGB and depth data, enhancing segmentation accuracy in cluttered and multi-layered object environments. We also introduce the Object Clutter Bin Dataset (OCBD), a benchmark dataset specifically tailored for evaluating bin-picking scenarios in top-down views. Extensive evaluations demonstrate that DA-Fusion outperforms state-of-the-art methods across diverse environments, making it particularly suited for real-world logistics tasks.

## 2. 论文提出的方法论
### 核心思想
采用 **可变形注意力（Deformable Attention）** 替代传统全图自注意力，在 RGB 和深度两个并行分支上分别进行**自模态精炼（Deformable Self-Attention, DS）** 和**跨模态融合（Deformable Cross-Attention, DC）**，从而在保持计算效率的同时捕捉局部重要区域和跨模态对应关系。

### 关键技术细节
- **输入处理**：RGB 图像 \(I_{rgb}\) 和深度图 \(I_d\)（转换为 3 通道 HHA 格式）。使用 4 层 Swin Transformer 作为 backbone 提取多尺度特征 \(f^i_{rgb}\) 和 \(f^i_d\)（\(i=1..4\)）。
- **Deformable Self-Attention (DS)**：
  - 对每个模态的特征图，基于查询 \(Q\) 学习一组偏移量 \(\Delta p\)，使注意力能够聚焦于变形后的采样点（K 个采样位置）。
  - 通过双线性插值在变形点处采样 key 和 value，计算多头注意力后得到精炼特征 \(f'_{rgb}, f'_d\)。
- **Deformable Cross-Attention (DC)**：
  - 利用 DS 输出的精炼特征，从 RGB 查询 \(Q'_{rgb}\) 预测朝向深度的偏移 \(\Delta p_{rgb \to d}\)，从而在深度特征上采样 key/value；同样从深度查询预测朝向 RGB 的偏移 \(\Delta p_{d \to rgb}\)。
  - 执行跨模态注意力（RGB query → depth key/value，depth query → RGB key/value），输出即为该层的融合特征 \(f_i\)。
- **解码**：4 层融合特征 \(f_1, f_2, f_3, f_4\) 拼接后送入 **Mask Transformer 解码器**（采用 Mask2Former 结构），生成最终的二值掩码 \(\hat{M} \in [0,1]^{Q \times H \times W}\)。
- **损失函数**：结合二元交叉熵损失和 Dice 损失（权重 \(\lambda_{ce}=5.0, \lambda_{dice}=5.0\)），再加上前景/背景分类损失（权重 \(\lambda_{cls}=2.0\) 对前景，0.1 对背景）。

### 公式与流程（文字说明）
1. DS 模块：\(f' = \text{DeformAttn}(f)\)，其中偏移由 \(Q = fW_Q\) 经过轻量网络预测。
2. DC 模块：\(f_i = \text{CrossDeformAttn}(f'_{rgb}, f'_d)\)，RGB 和深度互为 query/key-value。
3. 所有层融合特征拼接后进入 Mask Transformer 解码 → 掩码二值化。
