## 摘要
对缺失模态具有鲁棒性的多模态学习对于实际应用至关重要。现有方法主要关注模态间缺失（即整个模态缺失），而忽略了模态内退化（即模态存在但严重损坏）。实践中，这两种缺失类型经常共存，导致现有方法效果不佳。为解决此局限，我们提出通用不完整多模态学习（GIML），这是一种通过动态质量感知同时处理模态间缺失和模态内退化的统一框架。具体而言，GIML 将异构缺失模式建模为连续的模态信息退化，从而实现退化感知的自适应融合。为实现可靠的质量感知，我们引入噪声感知质量估计器，通过受控噪声注入学习从损坏特征到噪声强度的映射。此外，我们提出噪声-语义解耦模块，将语义信息与噪声干扰分离，从而提升对未见损坏模式的鲁棒性和泛化能力。在多种模态类型数据集上的大量实验证明了 GIML 的有效性和通用性。代码开源于：https://github.com/Yu-Five/GIML。

## Abstract
Multimodal learning robust to missing modalities is essential for real-world applications. Existing methods mainly focus on inter-modality missing, where entire modalities are absent, while overlooking intra-modality degradation, where modalities are present but severely corrupted. In practice, these two types of missing often coexist, making existing approaches ineffective. To address this limitation, we propose General Incomplete Multimodal Learning (GIML), a unified framework that simultaneously handles both inter-modality missing and intra-modality degradation through dynamic quality perception. Specifically, GIML models heterogeneous missing patterns as continuous modality information degradation, enabling degradation-aware adaptive fusion. To achieve reliable quality perception, we introduce a Noise-aware Quality Estimator that learns the mapping from corrupted features to noise intensity through controlled noise injection. Furthermore, we propose a Noise-Semantic Decoupled module that separates semantic information from noise interference. This improves robustness and generalization to unseen corruption patterns. Extensive experiments across datasets with diverse modality types demonstrate the effectiveness and generality of GIML. Code is available at: https://github.com/Yu-Five/GIML.

## 1. 论文的核心问题与整体含义

### 研究动机与背景
- **核心问题**：现有不完整多模态学习方法主要关注**模态间缺失**（整个模态完全缺失），而忽略了实践中更常见的**模态内退化**（模态存在但被严重损坏，如噪声、模糊、遮挡）。两者在实际系统中常共存（如传感器故障、环境干扰），但现有方法要么只能处理一种，要么采用两阶段串行处理导致优化冲突和次优性能。
- **整体含义**：本文提出一个**统一框架 GIML**，将两种缺失类型统一建模为**连续模态信息退化**，通过动态质量感知实现退化感知的自适应融合，从而同时处理模态间缺失和模态内退化。

## 2. 方法论

### 核心思想
- 将异构缺失模式（从轻微污染到完全缺失）视为连续的退化谱，用连续系数 `w_i^(v) ∈ [0,1]` 代替传统二元指示符 `δ_i^(v) ∈ {0,1}`，实现统一的模态质量建模。
- 通过两个专用模块实现可靠的质量估计和鲁棒语义表示：
  1. **噪声感知质量估计器 (NQE)**：直接学习从损坏特征到噪声强度的映射，提供准确的退化估计。
  2. **噪声-语义解耦模块 (NSD)**：将模态特征分解为语义均值（μ）和不确定方差（σ），显式分离语义信息与噪声干扰。

### 关键技术细节
- **NSD 模块**：
  - 对每个模态输出 `z_i^(v)`，通过两个分支 `f_μ` 和 `f_σ` 参数化高斯分布的均值和方差：`μ_i^(v) = f_μ(z_i^(v))`，`log σ_i^(v) = f_σ(z_i^(v))`。
  - 采用重参数化技巧采样：`s_i^(v) = μ_i^(v) + σ_i^(v) ⊙ ϵ, ϵ ~ N(0,I)`，保证可微。
  - 分类损失 `L_cls` 施加于语义均值 `μ` 上，确保保留判别信息；KL 散度正则化 `L_reg = KL[N(μ_i,σ_i²I) || N(0, η_i²I)]` 使预测方差 `σ_i²` 对齐真实退化强度 `η_i`，防止噪声混入语义。

- **NQE 模块**：
  - 轻量网络 `f_t` 从 NSD 得到的方差 `σ_i^(v)` 预测退化强度：`ˆη_i^(v) = f_t(σ_i^(v))`。
  - 使用 MSE 损失 `L_mse = (ˆη - η)²` 进行直接监督。
  - 基于逆方差原则计算融合权重：`ω_i^(v) = V·∑_{u≠v}(ˆη_i^(u))² / ∑_k (ˆη_i^(k))²`，退化严重的模态权重自动降低。

- **总体训练损失**：`L = L_cls^f + β₁∑ L_cls^(k) + β₂ L_reg + β₃ L_mse`。

### 算法流程说明（文字）
1. 对每个样本，随机抽取噪声强度 `η_i^(v)`，对原始输入施加噪声（如掩码、高斯）生成损坏输入 `x_i^(v)`。
2. 各模态编码器提取特征 `z_i^(v)`，经 NSD 模块分解为 `μ_i^(v)` 和 `σ_i^(v)`，采样得到语义表示 `s_i^(v)`；同时 `σ_i^(v)` 输入 NQE 预测退化强度 `ˆη_i^(v)`。
3. 融合权重由 `ˆη_i^(v)` 计算，加权融合得到最终表示，完成分类任务。
4. 联合优化所有损失项（分类、KL、MSE）。

1. GIML 在所有数据集和退化设置下**一致优于** TMDC 和 T2DR，尤其在严重退化或模态不平衡时优势更大（如 CREMA-D 上全可用时提升约 6% 绝对准确率）。

## 7. 优点
- **统一框架**：首次将模态间缺失和模态内退化统一建模为连续退化，避免两阶段串联的优化冲突。

- **强泛化能力**：NSD 显式分离语义与噪声，使得模型对未见噪声类型和强度具有天然鲁棒性。
- **广泛的实验验证**：覆盖多模态类型、多退化模式、多任务，结果全面且一致。
