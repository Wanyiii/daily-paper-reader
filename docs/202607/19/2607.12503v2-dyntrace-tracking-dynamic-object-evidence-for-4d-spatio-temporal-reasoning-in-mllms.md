## 摘要
4D时空推理，即联合建模3D空间结构和时间演化，对于理解动态世界和实现具身交互至关重要。尽管当前的多模态大语言模型（MLLM）在静态场景理解和粗粒度4D任务中表现出强大的能力，但在连续动态场景感知方面仍存在显著局限性，尤其是在追踪动态对象证据以进行连贯的4D时空推理方面。这一缺陷主要源于依赖稀疏的帧级观测，导致连续动态线索碎片化，使模型无法区分真实的物体动态与相机引起的表观运动。受人类在补偿视角变化的同时追踪动态线索的启发，我们提出了DynTrace，一个无需训练的4D时空推理框架，包含两个互补组件。动态轨迹可视化（DTV）将世界坐标轨迹重新投影到图像平面，提供基于几何信息的视觉先验，从而区分真实的物体动态与相机引起的表观运动。同时，动态痕迹令牌（DT-Token）组织成动态痕迹图（DTG），追踪对象级别的动态线索、痕迹演变和关键时刻，为连贯的4D推理维护连续的动态对象证据。这两个组件共同为MLLM提供了基于几何信息视觉先验和结构化时空痕迹的连续追踪动态对象证据。DynTrace持续改进开源MLLM，在Dyn-Bench、VLM4D和DSI-Bench上取得了最先进的结果，验证了追踪动态对象证据对于鲁棒4D时空推理的重要性。

## Abstract
4D spatio-temporal reasoning, jointly modeling 3D spatial structure and temporal evolution, is essential for understanding dynamic worlds and enabling embodied interaction. While current Multimodal Large Language Models (MLLMs) show strong capabilities in static scene understanding and coarse-grained 4D tasks, they still have notable limitations in continuous dynamic scene perception, especially in tracking dynamic object evidence for coherent 4D spatio-temporal reasoning. This shortcoming stems mainly from relying on sparse frame-level observations, fragmenting continuous dynamic cues and leaving models unable to disentangle genuine object dynamics from camera-induced apparent motion. Inspired by humans tracking dynamic cues while compensating for viewpoint changes, we propose DynTrace, a training-free framework for 4D spatio-temporal reasoning with two complementary components. Dynamic Trajectory Visualization (DTV) reprojects world-coordinate trajectories onto the image plane, providing geometry-informed visual priors that disentangle genuine object dynamics from camera-induced apparent motion. Meanwhile, the Dynamic Trace Token (DT-Token), organized into a Dynamic Trace Graph (DTG), tracks object-level dynamic cues, trace evolution, and key moments, maintaining continuous dynamic object evidence for coherent 4D reasoning. Together, these two components equip MLLMs with continuously tracked dynamic object evidence, grounded in geometry-informed visual priors and structured spatio-temporal traces. DynTrace consistently improves open-source MLLMs, achieving state-of-the-art results on Dyn-Bench, VLM4D, and DSI-Bench, validating the importance of tracking dynamic object evidence for robust 4D spatio-temporal reasoning.

  - **动态痕迹丢失（Dynamic Trace Loss）**：依赖稀疏帧采样或压缩的时序表示，割裂了物体轨迹的连续性，模型注意力漂移，丢失关键动态证据。
- **整体含义**：4D时空推理能力是具身智能、机器人交互、开放世界理解的基础。现有MLLM的瓶颈不在于视觉外观理解，而在于**未能持续追踪动态对象的证据**（dynamic object evidence）。受人类视觉认知中自动补偿视角变化并同时关注运动实体轨迹的启发，论文提出无需训练的DynTrace框架，通过显式构造连续动态证据来弥补这一差距。

### 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

- **核心思想**：将原始视频转换为两种显式的动态证据——**几何指导的视觉先验**（DTV）和**结构化的时空痕迹图**（DTG），在推理时直接馈入MLLM，使其无需从稀疏帧中隐式重建动态。
- **三个主要阶段**：
  1. **动态对象提取（Dynamic Objects Extraction）**  
     - 使用Qwen3-VL-8B根据问题 `q` 提取相关视觉实体描述集合 `E`。  
     - 使用WAFT（Warping-Alone Field Transforms）估计稠密光流，通过拟合背景仿射变换去除相机运动（Camera Motion Elimination），得到残差运动场。  
     - 对残差运动进行聚类（Dynamic Object Clustering），获得动态对象候选。  
     - 以 `E` 作为文本提示、聚类点作为点提示，输入SAM3（Segment Anything Model 3）得到时间一致的动态掩码 `M`。
  2. **时空动态编码（Spatio-Temporal Dynamics Encoding）**  
     - 使用Depth Anything 3（DA³）估计每帧深度、相机内参和位姿。  
     - 通过公式 `p_t^i = T_t^{c→w} ( z_t^i K_t^{-1} [u_t^i, v_t^i, 1]^T )` 将每个动态对象的中心点投影到世界坐标系，得到连续物体轨迹 `{p_t^i}`。  
     - 从相机位姿总结相机行为（Camera Behavior），并计算对象间距离和方位角得到关系演化（Relation Evolution）。  
     - **DTV（动态轨迹可视化）**：将世界坐标轨迹中的短探针点通过相机投影重新映射到图像平面（轨迹重投影，Trajectory Reprojection），并在对象掩码上绘制方向箭头，形成几何指导的视觉先验。  
     - **DTG（动态痕迹图）**：将视频划分为 `K` 个时间窗口，为每个窗口的对象生成DT-Token，包含动态线索（Cues）、痕迹演化（Trace Evolution）、关键时刻（Key Moments）。对象间关系也类似地生成关系令牌。所有令牌组织成窗口级图，串行化为文本形式的DTG。
  3. **表示集成与推理（Representation Integration & Reasoning）**  
     - 将用户查询 `q`、DTV集合 `D` 和串行化的DTG文本 `G` 一起输入目标MLLM，输出最终答案 `ŷ`。
- **关键技术细节**：
  - DTV通过世界坐标到图像平面的投影，使真实运动方向不随相机平移/缩放而变化，直接消除表观运动歧义。
  - DTG中的DT-Token包含三类信息：动态线索（当前位置、速度、朝向）、痕迹演化（窗口内趋势）、关键时刻（转弯、最近点等），使长距离连续轨迹显式可查。
  - 整个框架无需训练，仅需调用预训练模型（Qwen3-VL、WAFT、SAM3、DA³）进行推理。

- 追踪动态对象证据是解决MLLM在4D时空推理中失败的关键。

### 7. 优点

- **无需训练**：直接应用于现有MLLM推理阶段，无需微调，实用性强。
- **设计原理具体且有认知启发性**：明确针对人类视觉中“补偿自我运动并同时追踪实体”的耦合机制设计，理论动机清晰。
- **两个组件互补且可解释**：DTV提供视觉层面修正，DTG提供文本层面结构化痕迹，共同形成连续动态证据链。可视化结果可帮助理解模型决策。
- **广泛验证**：在三个不同侧重点的基准（对象交互、视角感知、观察者-物体解耦）上均取得一致提升，证明了方法的通用性。

### 8. 不足与局限

- **自我中心视角转换不足**：当问题需要将运动转换为对象自身的第一人称坐标系时（如“相对于角色自身方向如何移动”），DynTrace提供的证据仍以第三人称和相机-物体关系为主，不够充分。
- **部分级细节缺失**：对于需要追踪肢体局部运动（如“手相对于狐狸的位置”），DynTrace将对象整体建模，无法捕获精细部分运动。
- **计算开销未量化**：虽然无需训练，但推理过程中需要多次调用外部工具（光流、深度/位姿、分割、MLLM），实际延迟和资源消耗未报告，可能影响实时性。
- **骨干覆盖有限**：仅在8B/14B/32B开源模型上测试，未推广至商用闭源模型（如GPT-4V、Gemini）或更大模型（如70B）。
- **基准局限性**：三个基准均以英文VQA形式评测，未涵盖中文场景或具身动作生成等更复杂的4D任务。
- **潜在误差传播**：依赖光流、深度、分割等模型，这些模型在遮挡、快速运动、低光照下的误差可能影响最终证据质量，文中虽测试了GT对比，但实际应用中需考虑。
