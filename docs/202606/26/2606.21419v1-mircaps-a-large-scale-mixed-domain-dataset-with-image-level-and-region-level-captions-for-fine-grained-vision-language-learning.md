## 摘要
尽管视觉语言模型（VLM）最近取得了进展，但针对通用和基于闭路电视的视频监控系统的混合领域图像-标题数据集仍然有限。为了填补这一空白，我们引入了一个大规模多模态数据集，包含141,364张图像、981,947个图像级标题、1,742,264个区域级标题和1,391,779个边界框标注。每张图像平均有七个描述整体场景不同方面的图像级标题，以及每个标注边界框对应的七个区域级标题。这些互补的标题类型旨在帮助VLM学习细粒度的视觉属性，包括对象类别、估计大小、颜色、动作、状态和周围环境背景。我们在两个重要的下游任务（图像描述和目标检测）上展示了该数据集的有效性。实验结果表明，轻量级VLM，包括SmolVLM-256M-Instruct、BLIP、BLIP2和Qwen2.5-VL 3B-Instruct，可以使用我们的数据集进行有效微调。我们的数据集和代码公开于https://zenodo.org/records/20418601。

## Abstract
Despite recent progress in Vision-Language Models (VLMs), mixed-domain image-caption datasets for both general-purpose and CCTV-based video surveillance systems remain limited. To address this gap, we introduce a large-scale multimodal dataset comprising 141,364 images, 981,947 image-level captions, 1,742,264 region-level captions, and 1,391,779 bounding box annotations. Each image is associated with an average of seven image-level captions describing different aspects of the overall scene, as well as seven region-level captions for each annotated bounding box. These complementary caption types are designed to help VLMs learn fine-grained visual attributes, including object categories, estimated sizes, colors, actions, states, and surrounding environmental context. We demonstrate the effectiveness of the dataset on two important downstream tasks: image captioning and object detection. Experimental results show that lightweight VLMs, including SmolVLM-256M-Instruct, BLIP, BLIP2, and Qwen2.5-VL 3B-Instruct, can be effectively fine-tuned using our dataset. Our dataset and code are publicly available at https://zenodo.org/records/20418601.

## 2. 方法论

### 核心思想
- **多粒度描述**：同时提供图像级（场景整体）和区域级（每个标注对象）描述，形成互补监督。
- **对象中心化**：区域级标题聚焦单个对象的属性，包括估计大小、颜色、动作、状态、环境上下文。
- **被动式生成**：使用InternVL3.5-14B（14B参数）为所有图像生成标题，通过注入元数据（如鸟种、飞机型号、船舶类型）和基于检测框的类别信息动态构建提示（prompt），增强语义丰富度。

## 6. 主要结论与发现

1. **数据集有效性**：MIRCaps可帮助轻量VLM（特别是SmolVLM-256M和Qwen2.5-VL-3B）生成更多细粒度描述（如对象类别、动作、颜色、大小、上下文）。
2. **微调效果**：微调后模型倾向于生成更简洁、对象为中心的标题，与数据集风格对齐；但预训练模型因生成更长、更丰富的描述而获得更高的自动指标分数（BLEU、CIDEr等），而语义对齐指标（BERTScore、CLIPScore）两者相似。
3. **检测性能**：YOLOv12s在MIRCaps上达到mAP@50=68.9%，优于RT-DETR-L（63.6%）；但整体检测性能低于某些原始数据集上的专用模型（如DFire上81.2%），归因于**不完全标注**——原始数据集未标注全部15个类别实例。
4. **数据质量**：图像质量良好（NIQE=5.75, BRISQUE=21.08），无损坏图像；标题的CLIPScore平均28.56，区域级略高于图像级；人工评估中图像级标题平均78.96%，区域级57.17%，颜色和场景上下文表现最佳，大小和额外信息表现较差。
5. **类别分布**：呈现长尾分布（Person, Head占多数，Bus, Vehicle等稀少），反映真实监控场景。

## 8. 不足与局限

1. **边界框标注不完整**：原始数据集未完全标注15个类别中的所有实例，导致检测性能下降（如YOLOv12s mAP@50仅68.9%），且检测任务无法公平对比原始数据集结果。
