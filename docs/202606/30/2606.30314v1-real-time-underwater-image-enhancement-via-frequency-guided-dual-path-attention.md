## 摘要
实时水下图像增强对于移动水下摄影和自主机器人系统至关重要，这些系统在实际部署中通常需要在有限的计算资源下实现低延迟和紧凑模型。最近基于结构重参数化的超轻量级CNN满足了这些约束，但纯粹在空间域中操作，忽略了水下退化的频率敏感特性。为了解决这个问题，我们提出了一种轻量级UIE框架，该框架集成了两个关键组件：具有固定DCT先验的多分支可重参数化卷积（MBRConv-DCT），在训练期间注入结构化方向频率先验；以及频率引导双路注意力（FGDPA）模块，通过双路设计融合空间和频谱线索以进行自适应特征调制。两个组件都与结构重参数化完全兼容：卷积分支在重参数化后引入零额外推理成本，而注意力模块仅产生最小的计算开销。实验表明，我们的模型仅用4.23K参数和600+ FPS即实现了最先进的性能，在定量指标和视觉质量上均优于更大的方法。代码可在https://github.com/LethyZhang/FGDPA获取。

## Abstract
Real-time underwater image enhancement (UIE) is crucial for mobile underwater photography and autonomous robotic systems, where practical deployment typically requires low latency and compact models under constrained computational resources. Recent ultra-lightweight CNNs based on structural re-parameterization meet these constraints but operate purely in the spatial domain, ignoring the frequency-sensitive nature of underwater degradation. To address this, we propose a lightweight UIE framework that integrates two key components: a Multi-Branch Reparameterizable Convolution with Fixed DCT Priors (MBRConv-DCT) that injects structured directional frequency priors during training, and a Frequency-Guided Dual-Path Attention (FGDPA) module that fuses spatial and spectral cues via a dual-path design for adaptive feature modulation. Both components are fully compatible with structural re-parameterization: the convolution branch introduces zero additional inference cost after re-parameterization, while the attention module incurs only a minimal computational overhead. Experiments show our model achieves state-of-the-art performance with only 4.23K parameters and 600+ FPS, outperforming much larger methods in both quantitative metrics and visual quality. Code is available at https://github.com/LethyZhang/FGDPA.

- **背景**：水下图像受波长依赖的光吸收和散射影响，导致细节模糊、颜色失真、对比度低，严重影响下游任务（如目标检测、导航）。实时增强对于移动水下摄影、自主水下机器人（AUV）和嵌入式视觉系统至关重要，需要低延迟、小模型和低功耗。
- **现有方法不足**：传统物理先验方法泛化有限；基于Transformer等大模型虽效果好但参数量大（千万级），无法部署到资源受限的边缘设备。近年来基于结构重参数化的超轻量CNN（如RepVGG、MobileOne、MobileIE）实现了毫秒级延迟和极低参数量，但它们**仅在空间域操作**，忽略了水下退化的**频率敏感特性**（吸收改变全局光谱分布，散射衰减高频边缘）。这导致效果次优，出现颜色褪色、纹理恢复不全。
- **核心问题**：如何在超轻量模型（参数量<10K）中显式引入频率感知能力，同时保持零额外推理开销或极小计算增量，以提升增强质量。

# 2. 论文提出的方法论：核心思想与关键技术细节

## 2.1 整体框架

- 基于MobileIE的骨干网络，保留其Feature Self-Transform (FST)模块，加入两个新组件：
  - **MBRConv-DCT**：多分支可重参数化卷积，注入固定DCT先验。
  - **FGDPA**：频率引导双路注意力模块。
- 两者均与重参数化兼容：卷积分支在推理时完全合入标准3×3卷积，**零额外参数和FLOPs**；注意力模块仅在固定32×32网格上进行FFT，计算成本极小。

## 2.2 MBRConv-DCT
- **设计**：在原有MBRConv（包含3×3、3×1、1×3、1×1等平行分支）基础上增加一个**线性DCT分支**。
- **DCT分支实现**：
  - 使用**固定的非DC（直流）3×3 DCT基核**（编码水平、垂直、对角等方向频率模式），通过深度可分离卷积计算响应。
  - 响应经可学习的1×1点卷积投影到目标通道，并用标量门控（初始化为0，tanh激活）调节。
  - 所有核**零均值、单位ℓ2范数**，保证数值稳定性。
- **重参数化**：由于整个DCT分支是线性的，可解析地融合到等效3×3卷积核中。推理时与其他分支合并，**零额外开销**。

## 2.3 FGDPA（频率引导双路注意力）
- **频率路径**：
  - 将特征图下采样到固定32×32网格，进行2D FFT，取**对数压缩幅度谱** M = log(1+|FFT|)，以稳定动态范围并对抗分辨率变化。
  - 对M进行全局平均池化、归一化，与空间全局平均池化特征拼接，经1×1卷积+Sigmoid生成**频率引导的通道注意力图** Ac (C×1×1)。
- **空间路径**：
  - 在空间域取通道维最大值和平均值的拼接，经1×1卷积+Sigmoid生成**空间注意力图** As (1×H×W)。
- **融合**：使用可学习标量α, β, λ进行静态混合：
  - A = (1-λ)(αAc + βAs) + λ(Ac ⊙ As)
  - 最终调制特征 F' = A ⊙ F。
- **关键设计点**：FFT仅在32×32网格上进行，无需逆FFT，所有卷积为1×1，效率极高。
