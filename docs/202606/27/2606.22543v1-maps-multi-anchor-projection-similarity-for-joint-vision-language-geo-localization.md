## 摘要
人类通过整合视觉感知线索和语言语义推理来定位地点，形成既直观又有结构的场景理解。尽管现有的地理定位模型在跨视角和跨模态设置中取得了显著进展，但它们大多建立在点对点对齐的基础上，这对于联合视觉-语言查询来说是不充分的。在这样的查询中，视觉和文本线索并非简单地作为独立参考，而是共同定义一个用于定位目标的语义子空间。本文将联合图像-文本查询的视觉-语言地理定位（VLGL）形式化为一个多锚点几何对齐问题，并为此设置提出了一个统一框架。为了实现这一形式化，我们提出了多锚点投影相似性（MAPS），这是一种新的度量方法，它在高维空间中从视觉和文本查询特征构建一个锚点平面，并通过目标特征在该平面上的投影长度来度量相似性。与评估孤立成对关系的余弦相似性不同，MAPS捕获了目标特征与联合查询子空间之间的几何一致性，在检索过程中提供了更具判别性的排序标准。为了使学习到的表示与此几何结构一致，我们进一步引入了基于MAPS的对比损失，驱动目标特征朝向相应的锚点平面。所提出的框架、相似性度量和训练目标共同在VLGL中实现了最先进的性能。

## Abstract
Humans localize places by integrating perceptual cues from vision with semantic reasoning from language, forming a scene understanding that is both intuitive and structured. Although existing geo-localization models have made substantial progress in cross-view and cross-modal settings, they are largely built upon point-to-point alignment, which is insufficient for joint vision-language queries. In such queries, visual and textual cues do not simply act as independent references, but jointly define a semantic subspace for locating the target. In this paper, we formulate vision-language geo-localization (VLGL) with joint image-text queries as a multi-anchor geometric alignment problem and propose a unified framework for this setting. To realize this formulation, we propose Multi-Anchor Projection Similarity (MAPS), a new metric which constructs an anchor plane from visual and textual query features in a high-dimensional space and measures similarity by the projection length of the target feature onto this plane. Unlike cosine similarity which evaluates isolated pairwise relations, MAPS captures the geometric consistency between the target feature and the joint query subspace, providing a more discriminative ranking criterion during retrieval. To make the learned representation consistent with this geometry, we further introduce a MAPS-based contrastive loss that drives target features toward the corresponding anchor plane. The proposed framework, similarity metric, and training objective jointly yield state-of-the-art performance in VLGL.

- **意义**：人类定位时自然结合视觉感知和语言语义推理。本文旨在让机器具备类似能力，通过将视觉-语言地理定位（VLGL）形式化为**多锚点几何对齐问题**，提出更合理的相似性度量和训练框架，以提升联合查询下的定位精度和鲁棒性。

## 2. 论文提出的方法论

### 核心思想
- 将视觉查询特征 **v** 和文本查询特征 **t** 视为两个协调的锚点，在高维嵌入空间中定义锚平面 **P** = span([v, t])。
- 对于候选地理参考特征 **r**，通过其到锚平面的投影长度衡量与联合查询子空间的一致性，同时引入方向约束，确保投影落在视觉和文本共同指示的有效扇区内。
- 训练阶段使用基于相同几何的**MAPS 对比损失**替代传统余弦相似度对比损失，使表示学习与检索几何一致。

### 关键技术细节
1. **MAPS 得分计算（三个阶段）**
   - **锚平面构建**：收集 v 和 t 构成锚矩阵 A = [v, t]，其 Gram 矩阵 G_A = [1, S_vt; S_vt, 1]，行列式 det(G_A) = 1 - S_vt² 表示锚平面面积。
   - **投影一致性得分**：将 r 加入得增广矩阵 B = [v, t, r]，计算 det(G_{A,r})。正交距离 d²(r, P) = det(G_{A,r}) / det(G_A)。投影得分 s_proj = sqrt(1 - d²)。
   - **方向约束**：求解投影系数 α, β 满足 r 在锚平面上的投影 p = αv + βt。要求 α ≥ 0 且 β ≥ 0。若否，计算偏离角 θ_dev，施加指数衰减权重 W_dir = exp(-θ_dev²)。最终得分 S_MAPS = s_proj · W_dir。

2. **MAPS 对比损失**
   - 给定 mini-batch，用 S_MAPS 替换余弦相似度，计算 query→reference 和 reference→query 两个方向的 InfoNCE 损失，取平均。
   - 鼓励正样本具有高 MAPS 得分，负样本得分低。

## 7. 优点

- **方法创新**：将 VLGL 明确建模为多锚点几何对齐，引入投影一致性加方向约束，比点对点余弦相似度更符合联合查询的本质。
- **训练-推理一致性**：提出基于 MAPS 的对比损失，使学习目标与检索目标完全匹配，避免训练和推理不一致导致的性能损失。
- **高效实用**：MAPS 具有封闭形式解析解，计算仅需少量标量操作，易于集成到现有框架中。
- **充分验证**：在两个大规模数据集、多种参考模态、噪声和域迁移场景下进行了全面实验，消融分析严谨。
- **可解释性**：通过几何分析（联合一致性分布、有效投影比例、模态贡献）揭示了 MAPS 的工作机制。

## 8. 不足与局限

- **多锚点扩展性**：当前仅处理两个锚点（图像+文本），当出现更多模态或查询实例时如何定义稳定子空间尚未研究。
- **模态缺失与不一致**：若某个模态缺失或语义冲突，所定义的几何结构可能不稳定，影响鲁棒性。
