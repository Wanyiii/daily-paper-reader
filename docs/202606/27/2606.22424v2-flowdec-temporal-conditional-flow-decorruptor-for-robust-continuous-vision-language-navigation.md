## 摘要
连续环境中的视觉与语言导航（VLN-CE）要求智能体在未见过的场景中遵循自然语言指令。尽管大型模型（LMs）推动了VLN-CE的发展，但其性能仍因现实世界的视觉污染而严重下降，这是一个关键但尚未充分探索的领域约束。我们提出时间条件流去污染器（FlowDec），一种专为基于LM的VLN-CE设计的新型图像恢复框架。FlowDec集成了混合时间条件策略，将生成流路径与历史上下文对齐，并采用动作中心引导过滤来动态评估和整合输出。大量实验表明，FlowDec在导航准确性和生成延迟方面均优于最先进的去污染方法。我们的方法为在不可预测的现实世界条件下实现鲁棒具身导航建立了一个高效且稳健的范式。

## Abstract
Vision-and-Language Navigation in Continuous Environments (VLN-CE) requires agents to follow natural-language instructions in unseen scenes. While Large Models (LMs) have advanced VLN-CE, their performance remains severely degraded by real-world visual corruptions, a critical yet underexplored domain constraint. We introduce Temporal Conditional Flow Decorruptor (FlowDec), a novel image restoration framework tailored for LM-based VLN-CE. FlowDec integrates a hybrid temporal conditioning strategy to align the generative flow path with historical context and employs action-centroid guided filtering to dynamically assess and integrate outputs. Extensive experiments demonstrate that FlowDec outperforms state-of-the-art decorruption methods in both navigation accuracy and generation latency. Our approach establishes a robust, efficient paradigm for resilient embodied navigation in unpredictable real-world conditions.

### 2. 方法论
- **核心思想**：提出 **FlowDec**，一个 **模型无关、即插即用** 的视觉去污染模块，通过 **混合时间条件流匹配（Hybrid Temporal Conditional Flow Matching）** 和 **动作中心引导过滤（Action-Centroid Guided Filtering）** 实现高效、时序一致的图像恢复。
- **关键技术细节**：
  - **潜在条件流匹配（CFM）**：在 VAE 潜在空间建模，通过速度场网络 \(v_\theta(z_t, c_i, t)\) 学习从噪声到干净潜在的可逆 ODE 路径。
  - **混合时间条件策略**：定义三种条件 \(c_1, c_2, c_3\)：
    - \(c_1\)：仅使用当前帧的增强图像。
    - \(c_2\)：使用当前帧增强 + 前一帧增强与真实帧的差 + 动作标签（训练早期主导）。
    - \(c_3\)：使用当前帧增强 + 前一帧去噪输出与增强帧的差 + 动作标签（训练后期主导，桥接域差距）。
  - **动态条件权重**：训练中固定 \(c_1\) 权重 0.3，\(c_2\) 和 \(c_3\) 共享 0.7 并逐步交替（\(c_2\) 从 0.7 衰减至 0.14，\(c_3\) 从 0 增至 0.56）。
  - **动作中心（Action Centroid）**：在训练时，对每种原子动作（前进、左转15°等）收集真实帧差异的均值和方差，构建高斯分布 \(A_{gt,a} = \mathcal{N}(\mu_{gt,a}, \sigma_{gt,a})\)。

### 7. 优点
- **方法创新**：首次将时间条件流匹配与动作中心过滤结合，实现导航感知的去污染。
- **模型无关**：作为独立的视觉模块，可即插即用于任何 LM-based VLN 模型（验证了 NaVid 和 Uni-NaVid）。
- **时序一致性设计**：通过动作中心分析帧间变化，选择性利用历史信息，避免误差累积。
- **高效推理**：通过初始点加速和单步生成，比扩散方法快数倍。
- **全监督训练**：仅使用模拟数据集内的图像对训练，无需额外真实污染数据。

### 8. 不足与局限
- **清洁输入退化**：在干净图像上 SR 略有下降（表 5），归因于合成训练与真实图像的域差距，虽然影响小但需优化。
- **依赖动作标签**：动作中心构建和推理过滤需要导航模型输出的原子动作；若动作预测不准确，可能导致过滤失效。
- **腐蚀类型有限**：仅测试 12 种合成腐蚀，真实世界可能出现未覆盖的退化（如传感器串扰、动态遮挡）。
- **实验 backbone 范围**：仅验证了 NaVid 和 Uni-NaVid 两种，虽具代表性但未覆盖所有 LM-based VLN（如 NaVILA、InternNav）。
- **训练数据来源**：训练图像对采集自模拟器，可能存在模拟与真实的域偏移，虽真实实验验证但需更多场景测试。
- **未考虑动作误差传播**：过滤机制假设动作中心稳定，但长序列中动作误差累积可能影响引导效果。
